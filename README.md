# VIPmrgj — 个人主页 & 项目门户

> 一个网址，全部项目。个人主页 + 在线工具集，点击卡片即用。

## 🏠 访问

打开 `index.html` 即进入个人主页，向下滚动到「项目作品」，点击卡片即可使用对应工具。

部署到 GitHub Pages 后直接访问：
```
https://vipmrgj.github.io/<repo名>/
```

## 📦 项目结构

```
├── index.html            ← 门户入口（个人主页 + 项目卡片）
├── Main/                 ← 个人主页源文件
├── MEME creator/         ← Meme 3秒生成器（45种特效 + 89音效）
├── traffic-exchange/     ← 流量桥（开发者互助社区）
├── TextCleaner/          ← 文章清洗器（20+清洗功能）
├── FunSites/             ← 摸鱼乐园（9个HTML小游戏）
└── README.md
```

## ⚡ 特性

- **纯前端** — 所有项目均为单文件 HTML，无需构建、无需服务器
- **零门槛** — 浏览器打开即用，不上传数据，完全离线可用（流量桥除外）
- **分类筛选** — 主页项目区支持按「工具 / 娱乐 / 其他」分类浏览
- **按需加载** — 仅在点击卡片时加载对应工具，首屏仅 ~45KB
- **源文件独立** — 各项目均可脱离门户独立运行

## 🚀 本地运行

```bash
# 直接浏览器打开
open index.html

# 或启动本地服务器
python3 -m http.server 8080
# 访问 http://localhost:8080
```
